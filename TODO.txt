User stories
============
* Clients can send raw messages to server
o Clients can reveive raw messages from server
o Communication can occur in length-prefixed mode
o Communication can occur in delimiter-terminated mode 

Engineering tasks
=================
o All server connections get a unique id
o Server triggers client raw packages event when first time data arrives since last read
o One server client shall get new ClientReceivedRaw events although another's buffer isn't read
